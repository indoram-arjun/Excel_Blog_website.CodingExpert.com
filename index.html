Serial Numbers

Excel stores dates as serial numbers, which represent the number of days since January 1, 1900.

For example, January 1, 1900, is represented as 1, January 2, 1900, as 2, and so on.

This allows for easy date calculations and comparisons in Excel.

1. Serial Number using Drag

You can create serial numbers by dragging cells.

Steps:
  1. Enter 1 in the first cell
  2. Enter 2 in the second cell
  3. Select both cells and drag the fill handle down to generate more serial numbers
Result:
Excel automatically continues the series (3, 4, 5…).

2. Serial Number using Keys

You can use keyboard shortcuts to fill numbers.

Steps:
  1. Enter 1 in the first cell
  2. Select the range where you want numbers
  3. Press Ctrl + D (Fill Down)
Note :For a proper series, first create at least 2 numbers (1, 2), then use drag or fill.

3. Auto Update Serial Number

Automatically updates when data changes.

Formula:
=IF(A2<>"",ROW()-1,"")
  1. Enter 1 in the first cell
  2. Select the range where you want numbers
  3. Press Ctrl + D (Fill Down)
Explanation:
  • Shows serial number only if data is present.
  • Updates automatically when rows are added or removed.

4. Serial Number using SEQUENCE Function

Generates numbers instantly.

Formula:
=SEQUENCE(10)
Result:
Creates numbers from 1 to 10 automatically.

5. Serial Number using Ctrl Hold

Quick trick using mouse + keyboard.

Steps:
  1. Enter 1 in the first cell
  2. Press and hold Ctrl
  3. Click and drag down to select cells
  4. Release Ctrl
Result:
Excel creates a sequence (1, 2, 3…) automatically as you drag.

6. Serial Number (Alphabets) using CHAR Function

You can generate alphabet serial numbers.

Formula:
=CHAR(65)
Result:
A

Example (Auto Alphabet Series):

=CHAR(ROW()+64)
Result:
A, B, C, D…
Key Notes:
  • Drag method is the easiest
  • Formulas are best for automatic updates
  • SEQUENCE() is useful for modern Excel
  • CHAR() helps create alphabet series

Arithmetic Operators in Excel

Arithmetic operators are symbols used to perform basic mathematical operations in Excel.

They allow you to create formulas that can add, subtract, multiply, divide, and perform other calculations on your data.

1. Addition (+)

The addition operator is used to add numbers together.

Formula:
=A1 + B1
Result:
Adds the values in cells A1 and B1.

2. Subtraction (-)

The subtraction operator is used to subtract one number from another.

Formula:
=A1 - B1
Result:
Subtracts the value in cell B1 from the value in cell A1.

3. Multiplication (*)

The multiplication operator is used to multiply numbers together.

Formula:
=A1 * B1
Result:
Multiplies the values in cells A1 and B1.

4. Division (/)

The division operator is used to divide one number by another.

Formula:
=A1 / B1
Result:
Divides the value in cell A1 by the value in cell B1.

5. Exponentiation (^)

The exponentiation operator is used to raise a number to the power of another number.

Formula:
=A1 ^ B1
Result:
Raises the value in cell A1 to the power of the value in cell B1.

6. Modulus (%)

The modulus operator is used to return the remainder after dividing one number by another number.

Formula:
=A1 % B1
Result:
Returns the remainder after dividing the value in cell A1 by the value in cell B1.
Key Notes:
  • Use parentheses to control the order of operations.
  • Excel follows the standard mathematical order of operations (PEMDAS).
  • Be careful with division by zero, as it will return an error.
  • Use the AutoSum feature for quick addition of ranges.

UNIQUE Function

The UNIQUE function is used to extract unique (distinct) values from a range or list. It automatically removes duplicates.

Basic Syntax:
=UNIQUE(array, [by_col], [exactly_once])
Parameters:
  • array: The range or array from which to extract unique values.
  • by_col (optional): A logical value indicating whether to compare by column (TRUE) or by row (FALSE). Default is FALSE.
  • exactly_once (optional): A logical value indicating whether to return only values that appear exactly once (TRUE) or all unique values (FALSE). Default is FALSE.
Example:

To extract unique values from the range A1:A10:

=UNIQUE(A1:A10)
Result:
Returns a list of unique values from the range A1:A10, removing any duplicates.

IF Function in Excel

The IF function is a logical function that allows you to make decisions based on certain conditions. It returns one value if the condition is true and another value if the condition is false.

Basic Syntax:
=IF(logical_test, value_if_true, value_if_false)
Parameters:
  • logical_test: The condition you want to test (e.g., A1 > 50).
  • value_if_true: The value to return if the condition is true.
  • value_if_false: The value to return if the condition is false.
Example:

To check if the value in cell A1 is greater than 50:

=IF(A1 > 50, "Pass", "Fail")
Result:
Returns "Pass" if the value in cell A1 is greater than 50, otherwise returns "Fail".

IFS Function

The IFS function is a more advanced version of the IF function that allows you to test multiple conditions and return different values based on those conditions.

Basic Syntax:
=IFS(logical_test1, value_if_true1, logical_test2, value_if_true2, ..., default_value)
Example:

To assign grades based on scores:

=IFS(A1 >= 90, "A", A1 >= 80, "B", A1 >= 70, "C", A1 >= 60, "D", "F")
Result:
Returns "A" if the value in cell A1 is 90 or above, "B" if it's 80 or above, "C" if it's 70 or above, "D" if it's 60 or above, and "F" for anything below 60.
Key Notes:
  • Use the IF function for simple conditions and the IFS function for multiple conditions.
  • Ensure that the logical tests are ordered correctly, especially when using IFS.
  • Be mindful of the data types being compared in the logical tests to avoid errors.
  • Use nested IF functions for more complex scenarios, but consider using IFS for better readability.

IFERROR Function in Excel

The IFERROR function is used to handle errors in Excel formulas. It allows you to specify a value to return if a formula results in an error, such as #DIV/0!, #N/A, #VALUE!, etc.

Basic Syntax:
=IFERROR(value, value_if_error)
Parameters:
  • value: The formula or expression to evaluate.
  • value_if_error: The value to return if the formula results in an error.
Example:

To handle division by zero errors:

=IFERROR(A1/B1, "Error: Division by Zero")
Result:
Returns the result of A1 divided by B1. If B1 is zero, it returns "Error: Division by Zero" instead of showing the #DIV/0! error.

What is IF with AND / OR?

The IF function can be combined with AND and OR functions to evaluate multiple conditions within a single formula.

  1. AND returns TRUE only if all conditions are met
  2. OR returns TRUE if any condition is met
1. IF with AND Function

The AND function evaluates multiple conditions and returns TRUE only when all conditions are TRUE.

Syntax:
=IF(AND(condition1, condition2, ...), value_if_true, value_if_false)
Example:

To determine if a student passes based on their marks in three subjects:

=IF(AND(A1 >= 40, B1 >= 40, C1 >= 40), "Pass", "Fail")
Result:
Returns "Pass" if the values in cells A1, B1, and C1 are all 40 or above; otherwise, it returns "Fail".
2. IF with OR Function

The OR function evaluates multiple conditions and returns TRUE if at least one condition is TRUE.

Syntax:

=IF(OR(A1 >= 40, B1 >= 40, C1 >= 40), "Pass", "Fail")
Result:

Excel VLOOKUP Functions

What is VLOOKUP?

The VLOOKUP function in Excel is used to search for a value in the first column of a table and return a value from the same row in a specified column.

Basic Syntax:
=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
Parameters:
Parameter Description
lookup_value The value to search for in the first column of the table.
table_array The range of cells that contains the data to be searched.
col_index_num The column number in the table_array from which to return a value.
range_lookup TRUE for approximate match, FALSE for exact match.

1. VLOOKUP – Basic

To look up a value in a table and return a corresponding value:

=VLOOKUP(A1, B1:D10, 2, FALSE)
Result:
Searches for the value in cell A1 within the first column of the range B1:D10 and returns the corresponding value from the second column of that range. The FALSE argument ensures that it looks for an exact match.

2. VLOOKUP – Condition

To perform a VLOOKUP based on a condition:

=IF(A1 > 100, VLOOKUP(A1, B1:D10, 2, FALSE), "Value is too low")
Result:
Checks if the value in cell A1 is greater than 100. If it is, it performs a VLOOKUP to find the corresponding value from the range B1:D10. If the value in A1 is not greater than 100, it returns "Value is too low".

3. VLOOKUP – Range

To perform a VLOOKUP that returns a value based on a range of values:

=VLOOKUP(A1, B1:D10, 2, TRUE)
Result:
Searches for the value in cell A1 within the first column of the range B1:D10 and returns the corresponding value from the second column of that range. The TRUE argument allows for an approximate match, meaning it will return the largest value that is less than or equal to A1.

4. VLOOKUP – Dynamic (basic)

To make a VLOOKUP dynamic by using a cell reference for the column number:

=VLOOKUP(A1, B1:D10, {2,3,4}, FALSE)
Result:
Uses the MATCH function to dynamically determine the column number of "Column Name" in the range B1:D1 and performs a VLOOKUP based on that column number.

5. VLOOKUP – Dynamic (advanced)

To create a more advanced dynamic VLOOKUP that allows for both dynamic column selection and dynamic table range:

=VLOOKUP(A1, columns("Column Name"), FALSE)
Result:
Uses the MATCH function to dynamically determine the column number of "Column Name" in the range B1:D1 and performs a VLOOKUP based on that column number. The columns function allows for a dynamic table range based on the column name.

6. VLOOKUP with MATCH

To combine VLOOKUP with the MATCH function for more flexible lookups:

=VLOOKUP(A1, B1:D10, MATCH("Column Name", B1:D1, 0), FALSE)
Result:
Uses the MATCH function to dynamically determine the column number of "Column Name" in the range B1:D1 and performs a VLOOKUP based on that column number.

7. VLOOKUP + CHOOSE

To use the CHOOSE function to create a more flexible VLOOKUP:

=VLOOKUP(A1, CHOOSE({1,2}, B1:B10, D1:D10), 2, FALSE)
Result:
Uses the CHOOSE function to create a virtual table that combines columns B and D, allowing for a VLOOKUP that can return values from either column based on the specified column index.

8. SUM + VLOOKUP

To sum values based on a VLOOKUP result:

=SUM(VLOOKUP(A1, B1:D10, {2,3}, FALSE))
Result:
Uses the VLOOKUP function to return values from columns 2 and 3 of the range B1:D10 based on the lookup value in A1, and then sums those values together.

9. Automated VLOOKUP

To create an automated VLOOKUP that updates based on changes to the lookup value:

=VLOOKUP(A1, B1:D10, 2, FALSE)
Result:
The VLOOKUP will automatically update its result whenever the value in cell A1 changes, allowing for dynamic lookups without needing to manually refresh the formula.

10. Multiple Sheet VLOOKUP

To perform a VLOOKUP across multiple sheets:

=VLOOKUP(A1, Sheet2!B1:D10, 2, FALSE)
Result:
Searches for the value in cell A1 within the first column of the range B1:D10 on Sheet2 and returns the corresponding value from the second column of that range. The FALSE argument ensures that it looks for an exact match.

11. Nested VLOOKUP

To nest multiple VLOOKUP functions for more complex lookups:

=VLOOKUP(A1, B1:D10, VLOOKUP("Column Name", B1:D1, 0), FALSE)
Result:
The inner VLOOKUP finds the column number for "Column Name" in the range B1:D1, and the outer VLOOKUP uses that column number to return the corresponding value based on the lookup value in A1.

12. VLOOKUP + ROWS

To use the ROWS function to create a dynamic VLOOKUP that adjusts based on the number of rows in the lookup range:

=VLOOKUP(A1, B1:D10, ROWS(B1:D1), FALSE)
Result:
Uses the ROWS function to dynamically determine the column index for the VLOOKUP based on the number of rows in the range B1:D1, allowing for a more flexible lookup that can adjust as the table structure changes.

13. INDEX + MATCH

To use the INDEX and MATCH functions together as an alternative to VLOOKUP for more powerful lookups:

=INDEX(B1:D10, MATCH(A1, B1:B10, 0), MATCH("Column Name", B1:D1, 0))
Result:
The MATCH function finds the row number for the lookup value in A1 and the column number for "Column Name", and the INDEX function returns the value at that intersection in the range B1:D10. This combination allows for more flexible lookups than VLOOKUP, as it can search in any direction and is not limited to searching only the first column.

14. Reverse INDEX + MATCH

To use the INDEX and MATCH functions in reverse order for lookups based on column values:

=INDEX(B1:D10, MATCH("Column Name", B1:D1, 0), MATCH(A1, B1:B10, 0))
Result:
The MATCH function finds the column number for "Column Name" and the row number for the lookup value in A1, and the INDEX function returns the value at that intersection in the range B1:D10. This allows for lookups based on column values rather than row values, providing even more flexibility than a standard INDEX + MATCH combination.

HLOOKUP Function

1. HLOOKUP Basic

To use the HLOOKUP function to look up a value in the first row of a table and return a value in the same column from a specified row:

=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
Result:
Looks up the value specified in lookup_value in the first row of the table defined by table_array and returns the value in the same column from the row specified by row_index_num. The optional range_lookup argument determines whether to perform an exact match (FALSE) or an approximate match (TRUE).

2. HLOOKUP with Conditions

To use the HLOOKUP function with conditions to look up a value based on specific criteria:

=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
Result:
This formula allows you to perform a horizontal lookup based on specific conditions. The lookup_value is the value you want to find, table_array is the range of cells that contains the data, row_index_num is the row number from which to return the value, and range_lookup determines whether to look for an exact match or an approximate match.

3. HLOOKUP with Dynamic Range

To use the HLOOKUP function with a dynamic range to look up values in a table that may change in size:

=HLOOKUP(lookup_value, OFFSET(table_array, 0, 0, 1, COUNTA(table_array)), row_index_num, [range_lookup])
Result:
This formula allows you to perform a horizontal lookup with a dynamic range. The lookup_value is the value you want to find, table_array is the range of cells that contains the data, row_index_num is the row number from which to return the value, and range_lookup determines whether to look for an exact match or an approximate match.

4. HLOOKUP with MATCH Function

To use the HLOOKUP function in combination with the MATCH function to perform lookups based on dynamic row numbers:

=HLOOKUP(lookup_value, table_array, MATCH(lookup_value, header_row, 0), [range_lookup])
Result:
This formula allows you to perform a horizontal lookup using the HLOOKUP function in combination with the MATCH function. The lookup_value is the value you want to find, table_array is the range of cells that contains the data, header_row is the row that contains the headers for the columns, and range_lookup determines whether to look for an exact match or an approximate match.

5. HLOOKUP with ROW Function

To use the HLOOKUP function in combination with the ROW function to perform lookups based on the current row number:

=HLOOKUP(lookup_value, table_array, ROW() - header_row + 1, [range_lookup])
Result:
This formula allows you to perform a horizontal lookup using the HLOOKUP function in combination with the ROW function. The lookup_value is the value you want to find, table_array is the range of cells that contains the data, header_row is the row that contains the headers for the columns, and range_lookup determines whether to look for an exact match or an approximate match. The ROW() function dynamically calculates the row number based on the current position of the formula.

6. HLOOKUP with Multiple Criteria

To use the HLOOKUP function with multiple criteria to perform lookups based on multiple conditions:

=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
Result:
This formula allows you to perform a horizontal lookup using the HLOOKUP function with multiple criteria. The lookup_value is the value you want to find, table_array is the range of cells that contains the data, row_index_num is the row number from which to return the value, and range_lookup determines whether to look for an exact match or an approximate match. You can use additional functions like IF or AND within the lookup_value argument to specify multiple criteria for the lookup.

7. HLOOKUP with Dynamic Column Index

To use the HLOOKUP function with a dynamic column index to perform lookups based on a column number that may change:

=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
Result:
This formula allows you to perform a horizontal lookup using the HLOOKUP function with a dynamic column index. The lookup_value is the value you want to find, table_array is the range of cells that contains the data, row_index_num is the row number from which to return the value, and range_lookup determines whether to look for an exact match or an approximate match. You can use functions like MATCH or COLUMN within the row_index_num argument to dynamically determine the column index based on specific criteria or the current position of the formula.

Dget function

The DGET function in Excel extracts a single, specific value from a dataset (database) that matches one or more defined criteria. It is a powerful, often faster alternative to VLOOKUP or INDEX/MATCH because it allows for multiple, flexible, and non-adjacent criteria, including searching from right-to-left.

=DGET(database, field, criteria)
Explain :
  1. Database: The entire table range, including column headers (e.g., A1:D100 ).
  2. Field: The column to retrieve, indicated by the header name in quotes (e.g., "Sales" ) or a column number (e.g., 3 ).
  3. Criteria: A separate cell range that contains at least one column label and a corresponding condition (e.g., F1:F2 ).
